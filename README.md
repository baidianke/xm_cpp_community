## c++社区

### 建议使用 yarn

### 安装依赖后，需配置 git pre-commit hook (很重要！！！)

```
yarn install && yarn run parpare
```

### 本地运行

```
yarn dev
```

```
yarn dev:test
```

#### 环境配置文件

env.config.js

#### 注意事项

若要使用 yarn server:\*\* ，需本地安装[pm2](https://pm2.keymetrics.io/docs/usage/quick-start/)
